# Milad Hemmati – Portfolio Website

Welcome to my personal portfolio website showcasing my UI/UX design work, CV, and multilingual pages.

## 📁 Structure

- `/index.html`: English homepage (default)
- `/indexDu.html`: Dutch homepage
- `/projects.html`: English project portfolio
- `/projectsDu.html`: Dutch project portfolio
- `/cv.html`: Curriculum Vitae
- `/Asset/`: Images, icons, language flags, etc.
- `/style*.css`: Individual CSS files for each page and language

## 🌐 Languages
- English (default)
- Dutch (via `/indexDu.html` and `/projectsDu.html`)
